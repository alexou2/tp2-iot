# frontend

1. download vue.js https://vuejs.org/
2. start the server
   1. cd frontend/
   2. npm run dev
3. open http://localhost:5173/

# raspberry pi code

1. install python
2. import tkinter, adafruit_dht, board, time, gpiozero and RPi.GPIO
3. run the code
   1. cd raspberry\ pi/
   2. python3 tp2.py
